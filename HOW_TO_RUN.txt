Employee Payroll Management System - How To Run
================================================

1. Run the Spring Boot web application
--------------------------------------
Open a terminal in:

EmployeePayrollSystem

Run:

mvnw.cmd spring-boot:run

The app starts an embedded Tomcat server and opens the browser automatically when supported.

Open this URL:

http://localhost:8080/login.html

3. Database
-----------
The project uses PostgreSQL for persistence. Configure your database connection in
`src/main/resources/application.properties` or preferably via environment variables.

Recommended (secure) approach:

1. Create a dedicated database and user (example commands shown below).
2. Set environment variables before running the app:

PowerShell example:

```powershell
$env:JDBC_DATABASE_URL = 'jdbc:postgresql://localhost:5432/payrolldb'
$env:JDBC_DATABASE_USERNAME = 'payroll'
$env:JDBC_DATABASE_PASSWORD = 'your_password'
.\\mvnw.cmd spring-boot:run
```

Linux/macOS example:

```bash
export JDBC_DATABASE_URL='jdbc:postgresql://localhost:5432/payrolldb'
export JDBC_DATABASE_USERNAME='payroll'
export JDBC_DATABASE_PASSWORD='your_password'
./mvnw spring-boot:run
```

Example psql commands to prepare the database (run as a superuser):

```sql
-- connect as postgres superuser
CREATE DATABASE payrolldb;
CREATE USER payroll WITH ENCRYPTED PASSWORD 'your_password';
GRANT ALL PRIVILEGES ON DATABASE payrolldb TO payroll;
```


4. If port 8080 is busy
-----------------------
Run on another port:

mvnw.cmd spring-boot:run "-Dspring-boot.run.arguments=--server.port=9090"

Then open:

http://localhost:9090/login.html


5. Build a runnable jar
-----------------------
mvnw.cmd -DskipTests package

Run the jar:

java -jar target/employee-payroll-system-1.0.0.jar
